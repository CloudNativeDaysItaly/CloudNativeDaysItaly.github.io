Cosa succede quando il "contributo" di un'intelligenza artificiale diventa un attacco DDoS per un maintainer? Benvenuti nell'era dell'AI Slop: un'ondata di Pull Request di basso valore e report di sicurezza del tutto inventati che sta travolgendo l'Open Source.

In questo lightning talk, vedremo come giganti della community come Daniel Stenberg (curl) e Mitchell Hashimoto (Ghostty) si siano ritrovati a lottare contro una marea di false CVE che allucinano vulnerabilità inesistenti o PR che risolvono problemi mai nati.
