In this hands-on workshop, participants will learn how it is crucial to work with hardened images early in the SDLC.
Through this workshop, attendees will gain a deep understanding of Docker Hardened images and how to use them in their development projects.
From using the community tiers to understanding what Enterprises need to apply in order to secure their development pipeline.

Key topics covered will include:
- Introduction to container and cloud native security
- Deep dive into the Docker Hardened Images and helm charts catalog
- Leverage a vulnerability management solution to discover what CVEs are impacting their application
- Start from an existing application and migrate the images used to hardened images to secure this application

By the end of the workshop, participants will have practical experience building secured containerized applications by applying security using images free of CVEs.
