Migrating an enterprise API gateway from a legacy system to Kubernetes has no consolidated playbook. In this talk, we share a real-world battlefield report: building a hybrid API platform on Kubernetes — with self-hosted data planes running in both on-premises and cloud environments — starting from a commercial legacy gateway in production.
