Kubernetes platforms often struggle with configuration drift and inconsistent environments across clusters. What if the operating system itself was immutable and fully controlled through declarative configuration? In this session, maintainers of the SIGHUP Distribution will share how we designed an immutable Kubernetes platform built on top of Flatcar Container Linux. The session will include a live demo showing how we bootstrap a cluster from a PXE boot to a fully operational immutable Kubernetes cluster.
